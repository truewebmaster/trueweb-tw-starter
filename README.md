# TW Starter

Персональная стартовая тема WordPress для разработки сайтов TrueWeb.

---

## Цель проекта

Создать собственную стартовую тему, которая ускоряет разработку новых проектов и развивается вместе с опытом.

---

## Технологии

- WordPress
- PHP
- SCSS
- JavaScript
- Git
- LocalWP
- SFTP
- VS Code

---

## Структура проекта

```
assets/
inc/
template-parts/
```
